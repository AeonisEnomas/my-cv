## Features

### Design & UI
- **Modern Dark/Light Theme** - Seamless toggle with CSS custom properties
- **Fully Responsive** - Optimized for mobile, tablet, and desktop
- **Smooth Animations** - Scroll reveals, typing effect, hover transitions
- **Professional Typography** - Inter font family with perfect hierarchy
- **Visual Skill Bars** - Animated progress bars for technical skills
- **Timeline Design** - Vertical timeline for education section

### Functionality
- **Typing Animation** - Dynamic rotating titles
- **Scroll Progress** - Top progress bar indicator
- **Sticky Navigation** - Auto-hiding desktop nav on scroll
- **Mobile Navigation** - Slide-out menu with overlay
- **Contact Form** - Real-time validation with toast notifications
- **Download CV** - Print-friendly PDF generation
- **Smooth Scrolling** - Smooth scroll to sections with offset

### Accessibility
- Keyboard navigation support
- Reduced motion support (`prefers-reduced-motion`)
- High contrast mode support
- Semantic HTML5 structure
- ARIA labels

## Technologies

- **HTML5** - Semantic markup
- **CSS3** - Grid, Flexbox, Custom Properties, Animations
- **JavaScript (ES6+)** - Modules, Intersection Observer, Fetch API
- **Font Awesome 6** - Icons
- **Google Fonts** - Inter & JetBrains Mono
