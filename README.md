# Abdullah Mushtaq - Portfolio

A professional portfolio and ready-to-send CV site built with React and Vite.

## Stack

- React 19
- Vite 7
- lucide-react icons
- CSS custom properties for responsive light and dark themes
- Static PDF asset for direct CV download

## Features

- Professional landing page with clear positioning and contact actions
- Employer-focused project descriptions
- Responsive layout for desktop, tablet, and mobile
- Light/dark theme toggle with saved preference
- Direct PDF download at `public/Abdullah-Mushtaq-CV.pdf`

## Project Structure

```text
my-cv/
|-- index.html
|-- package.json
|-- vite.config.js
|-- public/
|   `-- Abdullah-Mushtaq-CV.pdf
|-- src/
|   |-- main.jsx
|   `-- styles.css
`-- Images/
    `-- Profile.JPG
```

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build is generated in `dist/` and can be deployed to Vercel, Netlify, GitHub Pages, or any static hosting provider.
